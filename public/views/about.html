About

Readly provides an easy way for users to access old articles.

Are there any advanced tagging options?

Add #readly + [time value] + [time unit]. The following time units are available.

Letter Time
h Hours
d Days
w Weeks
m Months
y Years

The default time schedule is 3 days. If no time units and value is specified (#readly) or is unrecognizable, the default time schedule is used.


Is there any way to set up repeated reminders?

As of now, no. But this feature is definately in the development roadmap.


What's Readly's privacy policy?

I won't use any of your information other than the sole purpose of sending out the requested links.


Who's behind Readly?

Readly is an one-man project where everything is designed and programmed by @smashthewindow. I'm always looking for ways to provide users with smooth and enjoyable reading experience - if you have any suggestions, come drop me a message!
